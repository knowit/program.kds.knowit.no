F# : what's in it for me?

Toni Kaihola - Knowit OY Group

Long talk (60 min), English, Friday 13:00-14:00 slot - room C3

Join me on a whirlwind tour into the functional cousin of C#. We will cover many areas of the language, including the powerful type system and useful language constructs. You will get answers to the following questions: How can you code without nulls? What if you could do calculations with units of measurements such as m/s ? Using extension methods without extension methods, how? How can you get value equality for free? I have a lot of existing C# or VB code. Can I somehow benefit from F#? Finally we will explore tools that you can use to get acquainted with F# scripting and tools which you can use with your existing .NET projects and beyond.

Managing your containers with Kubernetes

Ole Bakstad - Knowit Objectnet

Long talk (60 min), English, Friday 13:00-14:00 slot - room 25

During the last few years containers have become a common word in the tech vocabulary. There are many providers that offer solutions for managing containers for example Google Cloud Platform, Amazon AWS and Mesosphere. This talk will introduce Kubernetes from Google, a orchestration tool that offer automated deployment, scaling and management of your containers. You will learn the core concepts and features of Kubernetes and gain insight in how it differs from similar solutions like Amazon Container Service or Mesosphere. I will also give a live demo of these features. The demo will show how to easily load balance between multiple container instances, auto scale your application and how to do canary releases and rolling upgrades with zero down time.

Building a healthcare service bus

Don Janssen - Knowit Norrland

Long talk (60 min), English, Friday 13:00-14:00 slot - room 26

In a hospital lots of different systems are used and usually they do not work very well together. At the same time systems need to share information with each other and patients and authorities want information regardless of the system that stores the information. To solve this problem we are building a service bus for the county of Västerbotten, based on principles for domain driven architecture, SOA, REST and MicroServices, using .Net and Web API. The service bus handles routing, service discovery and has an engagement index that keeps track of patients and in which system they have data. Although every service is independent of the other a lot of functionality is the same for every service, therefore we created a framework existing of NuGet packages. As part of the project we also build a web portal on top of the service bus that makes the information about the patients accessible for the hospital's employees. In this talk I will discuss the challenges but most important the solutions we are working on to solve those integrations.

HTTP/2 solves many issues but it also introduces some changes that we should know about. By the way, HTTP/2's official symbol is H2 (according to Surma @ Google) so you can go ahead and refer to HTTP/2 as H2.

Solving the mysteries of the universe with parallel programming

Andreas Hunderi - Knowit Experience Bergen

Short talk (30 min), English, Saturday 09:40-10:10 slot - room 24

How can we harness the power of parallelism to tackle complex problems that can advance our understanding of the universe? The N-body problem is a collection of problems of how objects or particles interact with each other given an initial speed, a direction and a force that acts upon all other ojects (for instance gravity). This is relevant in many aspects of our lives today, like astrophysics, medicine or the simulation of weather patterns. I plan to talk about the N-body problem, and show what a serial and parallel implementation of the problem would look like.

MLaaS - Machine Learning for the rest of us

Anders Breivik - Knowit Objectnet

Short talk (30 min), English, Saturday 09:40-10:10 slot - room 25

What is Machine Learning as a Service? Which do I choose? How can I use it in my application? Machine learning has long been a somewhat academic discipline, its practitioners highly specialized. But with the advent of a new class of commoditized machine learning platforms, MLaaS, Machine Learning as a Service, is now available to all developers. MLaaS can be treated as a black box, giving developers the opportunity to dive into machine learning head first. Supply your own data, build a model using tools supported by your chosen MLaaS platform and train it using the well tested algorithms offered. When ready to dig deeper into the world of machine learning, most MLaaS platforms enables us to develop our own algorithms even better tailored to our data and models. This talk will show you the state of the art of MLaaS, what the different vendors offers, and where you best can put it to use in your applications.
