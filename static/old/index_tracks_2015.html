Knowit Developer Summit

13. - 14. november

Call for papers

Vi planlegger Knowits årlige felles utviklerkonferanse! Denne blir avholdt 13. - 14. november 2015 i Stockholm, Sverige. Siste frist for å sende inn bidrag er 3. august 2015.

I den forbindelse leter vi etter motiverte utviklere til å snakke om emner som:

Arkitektur
Mikrotjenester, DDD, containere (Docker, Rocket...), event-sourcing, asynkronitet, feiltolerante systemer, REST, dependency injection, web services uten rammeverk, store systemer uten app servere, in-memory computing
Mobil
Mobil først-design, testverktøy, multi-plattform utvikling, Apple Watch, iOS, Android, Windows Mobile
Devops
Infrastruktur som kode, provisioning, kontinuerlig deployment
Persistens
NoSQL, SQL, big data, cloud, grid
Web
*.js, responsivt design, HTML5, Less/Sass
Testkode
Skrive gode tester, testverktøy, kontinuerlig testing, TDD, BDD
Programmering
Paradigmer, parallelisering, typing, språk (C#, F#, Scala, Elixi, Clojure, etc.)
Verktøy
Versjonskontroll, byggesystemer, SaaS, analyseverktøy(Sonar, R, o.l.)
Plattformer og annet
Cloud, Internet of Things, CMS, CRM, Office 365, EPiServer

... og mer!

Ditt publikum? Lidenskapelige og interesserte systemutviklere i Knowit, hovedsaklig innen Java- og .NET plattformene.

Vi ønsker oss forslag til workshops, fulle foredrag (60 min), mindre foredrag (30 min) og lyntaler (10 min).

Vi er interessert i foredrag og andre bidrag som vil inspirere og engasjere deltagerne!

Mer informasjon om lokasjon, påmelding og andre praktiske detaljer kommer senere. Hvis du har noen spørsmål ta kontakt på kds@knowit.no.
